About Project

This project is about a Bulletin Board that is an updated better version of the Homebase Digital Message Center released by Audiovox in 2008. The Bulletin Board is basically a screen on the wall for all ages who are able to communicate or are able to use the device. This was created for communication, security, and organization in a household. Prototype created in Adobe XD.

Features of Project

  • Facial recognition- This function is for a hands-free approach to using the bulletin board. Also, for more security.
  • Voice recognition- This function is also for a hands-free approach and for security. Also, for using the board wherever you are in the house.
  • Connect to devices- This allows any device to be connected to the board including phones, TVs, tablets, and security cameras.
  • Warn users of danger outside or inside the home- This is a big security feature where the user is kept safe from intruders or other dangers.
  • Warn users of danger in the neighborhood- This is also a big security feature where the user is aware of danger in the neighborhood to stay away from the area and be aware of danger.
  • Knows user schedule and tells them what time to leave based on traffic- This helps with user schedule to keep the user on time to events and other things on the schedule.
  • Alerts user of traffic- This feature warns the user of more or less traffic based on where they are going and when they would need to leave to be on time.
  • Calendar- This feature allows users to add appointments and reminders and is shareable between users. It alerts users if user sets to alert before a reminder and an appointment.
  • Messages- This feature allows users to send text messages, video messages, and call others users. It also allows users to leave these messages for a user when they come home or login to their connected device away from home.
